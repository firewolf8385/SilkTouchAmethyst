# SilkTouchAmethyst &middot; [![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](http://makeapullrequest.com) [![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg?style=flat-square)](https://github.com/JadedMC/CommandBlocker/blob/master/LICENSE)

SilkTouchAmethyst is a simple plugin that allows players to collect Budding Amethyst blocks by mining them with Silk Touch. No configuration, no commands, just plug and play!

    <a href="https://modrinth.com/plugin/silktouchamethyst"><img alt="modrinth" height="40" src="https://cdn.jsdelivr.net/npm/@intergrav/devins-badges@3/assets/cozy/available/modrinth_vector.svg"></a>
</p>

## Requirements
* Java 17 or newer
* Server Software with Bukkit API.
